Privacy Notice

Plain-language summary — this is a small self-hosted practice tool, not a commercial product.

This practice test doesn't create online accounts, and it doesn't send names, answers, or scores to any central server or database.

Everything — student name, answers, flags, section timing, and past-attempt history — is saved only inside the web browser on whatever device is being used, using the browser's built-in storage. Clearing that browser's site data, or switching to a different browser or device, means that saved progress won't carry over.

The hosting provider (e.g. GitHub Pages) may keep its own standard technical logs, such as IP address, browser type, and access time, under its own terms — this site has no visibility into or control over those logs.

For parents and teachers

Consider using a nickname, initials, or a code instead of a child's full legal name in the "Student name" field — it's shown on-screen and saved in the browser, but isn't needed for the practice test to work.

Please don't enter email addresses, phone numbers, home addresses, or other sensitive personal information anywhere in this tool — none of the fields are designed to hold that kind of data, and it isn't needed.

If multiple children share the same browser or device, one child's saved progress can be overwritten by another's. Use the "Print results" button after a session to keep a record before someone else uses the same device.

Practice stanine scores

Any "practice stanine" shown on the results screen is an estimate calculated from percent-correct on this specific practice test, and only appears after a full test is completed. It is not an official ERB/ISEE stanine — real ISEE stanines are norm-referenced by grade using a national pool of test-takers, which this site has no access to. Treat it only as a rough way to track practice trends over time.
